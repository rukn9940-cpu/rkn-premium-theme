# RKN Premium Theme

**RKN Premium Theme** is a commercial [Salla](https://salla.sa) storefront theme
for the **Salla Theme Store**, built on the Twilight theme engine. It ships a
complete storefront, customer account area, blog, and campaign pages — with an
advanced design system, premium shopping features, Dark Mode, full RTL/LTR
support, WCAG 2.2 AA accessibility, and Core Web Vitals-optimized delivery.

Every visible element is driven by **merchant Theme Settings** or **live Salla
data** — no bundled demo products, stock photos, sample banners, or placeholder
content.

- **Repository:** [github.com/rukn9940-cpu/rkn-premium-theme](https://github.com/rukn9940-cpu/rkn-premium-theme)
- **Support:** [GitHub Issues](https://github.com/rukn9940-cpu/rkn-premium-theme/issues)

## Requirements

- Node.js `>= 18` (see `.nvmrc`)
- pnpm `>= 9`
- [Salla CLI](https://www.npmjs.com/package/@salla.sa/cli) for theme preview/sync:
  ```bash
  npm install -g @salla.sa/cli
  salla login
  ```

## Getting started

```bash
# 1. Install dependencies
pnpm install

# 2. Production build (minified CSS/JS → public/)
pnpm production

# 3. Development watcher (rebuilds on change, syncs with Salla preview)
pnpm watch

# 4. In another terminal, start the Salla theme preview
salla theme preview
```

## Scripts

| Script              | Description                                           |
| ------------------- | ----------------------------------------------------- |
| `pnpm dev`          | One-off development build (unminified, source maps).  |
| `pnpm watch`        | Development build that rebuilds on file changes.      |
| `pnpm production`   | Optimized production build (minified CSS/JS).         |
| `pnpm lint`         | Lint JavaScript and SCSS.                             |
| `pnpm format`       | Format JS/SCSS with Prettier.                         |

## Homepage components

All merchant-configurable from the Salla home-page builder (`twilight.json`):

| Component | Description |
| --- | --- |
| Hero Banner | 10 layout variants, merchant-uploaded slides |
| Store Categories | Real Salla categories |
| Featured Products | Real Salla products |
| Promotional Offer | Merchant image + optional countdown |
| Brand Showcase | Real Salla brands |
| Marketing Text Block | Merchant-authored copy |
| Image Gallery | Merchant-uploaded images |
| Promo Video | Merchant YouTube URL |
| Product Reels | Merchant videos linked to products |
| Banner Builder | Unlimited promo banners (full/half/third) |

Salla Theme Features also power **Testimonials** and **Store Features** sections
using real store data configured in the Partners Portal.

## Feature highlights

- **10 color presets** + full design system (radius, shadows, spacing, font
  scale, button shape) — all Theme-Settings-driven, zero rebuild.
- **8 header layouts, 8 footer layouts, 10 hero layouts, 5 product card styles**
- **Premium shopping:** Product Reels, 360° viewer, Mega Menu, Live Search,
  Quick View, Compare, Sticky ATC, Recently Viewed, Smart Recommendations
- **Dark Mode**, RTL/LTR, animations, SEO metadata, JSON-LD structured data
