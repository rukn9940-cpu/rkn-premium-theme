# Changelog

All notable changes to **RKN Premium Theme** are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-07

### Added

- Initial commercial release for the Salla Theme Store.
- Full storefront: home, product listing/single, cart, thank-you, brands, blog,
  static pages, landing/campaign pages, and loyalty program.
- Customer account area: profile, orders, wishlist, and notifications.
- Premium shopping features: Product Reels, 360° viewer, Mega Menu Builder, Live
  Search, Quick View, Compare Products, Sticky Add to Cart, Recently Viewed,
  and Smart Recommendations — all merchant-configurable from Theme Settings.
- Design system: 10 color presets, 8 header layouts, 8 footer layouts, 10 hero
  layouts, 5 product card styles, Banner Builder, radius/shadow/spacing/font
  scales, and button shapes.
- Dark Mode, RTL/LTR support, WCAG 2.2 AA accessibility baseline, SEO metadata,
  and Core Web Vitals-optimized asset delivery.

### Removed

- All Theme Raed reference artifacts (`tree.json`, Raed-specific comments).
- Bundled demo imagery (`placeholder.svg`) and dev-only Twig drafts.
- Hardcoded sample/demo content — every visible element is driven by merchant
  Theme Settings or live Salla API data.

### Changed

- Repository, support URL, and commercial metadata aligned for
  [github.com/rukn9940-cpu/rkn-premium-theme](https://github.com/rukn9940-cpu/rkn-premium-theme).
- Package manager standardized on **pnpm**.
- Component titles and descriptions polished for Salla Theme Store listing.
